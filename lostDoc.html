Find Lost Documents Application

An Academic Project to Learn How to Connect Java with a Database.

Process Highlights

Project Overview

The Find Lost Documents Application is a document recovery system that helps users locate misplaced files by querying metadata (filename, date, type) stored in a relational database. Users can efficiently search for lost documents using intuitive filters, ensuring quick retrieval while maintaining data integrity. The system leverages Java and Oracle for robust backend functionality.

Challenge

Develop a reliable search system with efficient database queries and accurate metadata retrieval while ensuring security and performance.

Opportunity

Apply Java and Oracle skills to build a structured, database-driven solution with real-world utility.
