Binary Research Trees Algorithm

An Academic Project to Learn How to Implement AVL Trees and Map Trees and use them.

Process Highlights

Project Overview

The Binary Research Trees project is an academic implementation of binary search trees in C++, featuring core operations such as insertion, deletion, and search. The project includes a shell-based visualization to demonstrate tree operations, helping users understand the underlying data structure and its behavior.
